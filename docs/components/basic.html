基础组件

基础表单组件

Mixin

  • mixins/basic.js 提供了基础表单组件基本功能。所以基础组件一般只需
    module.exports = {
mixins: [basicMixin]
};

  • mixins/options.js 提供了select、checkboxs等多选表单元素的options异步设置功能。在表单元素上，我们暴露了data-set-options方法，所以

    ready: function() {
setTimeout(function() {
  var options = {
    '【投放位】推品': '0',
    '【投放位】商加品': '1',
    'offer加强版': '2'
  };

  $('select[name="source"]').data('setOptions')(options);
}, 1000);
}
